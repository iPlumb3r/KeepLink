Concepts de base
-
KL utilise assentiellement les 3 __concepts__ suivants : 
* l'__Item__
* Le __Lien__
* Et le __Type__

Le concept de __Type__ pouvant se combiner avec __Item__ et __Lien__, on obtient les _concepts_  suivants : 
* Le __Type d'Item__
* Le __Type de Lien__
